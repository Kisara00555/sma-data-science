# 🧬 SMA Data Science Project

## 📌 Overview
This project applies data science techniques to analyze **Spinal Muscular Atrophy (SMA)**, a rare genetic disorder that affects motor neurons and muscle function.

The goal is to explore clinical and genetic features and build machine learning models to predict disease severity.

---

## 🧠 About SMA
Spinal Muscular Atrophy (SMA) is a genetic condition caused by mutations in the SMN1 gene, leading to reduced production of SMN protein.

The number of SMN2 gene copies influences disease severity:
- More SMN2 copies → milder disease
- Fewer SMN2 copies → more severe disease

---

## 📊 Dataset
This project uses a **synthetic clinical dataset** created for educational purposes.

Features include:
- Age
- SMA type
- SMN2 copy number
- Motor function score
- Disease severity

---

## ⚙️ Project Workflow
1. Data Loading  
2. Data Cleaning  
3. Exploratory Data Analysis (EDA)  
4. Feature Engineering  
5. Machine Learning Modeling  
6. Model Evaluation  

---

## 🤖 Models Used
- Logistic Regression  
- Decision Tree  

---

## 📈 Results
- Both models achieved an accuracy of **0.50**
- Strong correlations observed between genetic and clinical features
- Feature importance analysis showed high reliance on age (due to dataset limitations)

---

## ⚠️ Limitations

- Synthetic (not real patient data)  
- Possible overfitting
